# Movie Search

A movie discovery application for searching and browsing films via the TMDb API, featuring real-time data fetching.

## Tech Stack

* **Framework**: React
* **UI Library**: Material UI
* **State Management**: TanStack Query
* **API**: The Movie Database (TMDb)

## Getting Started

1. **Install dependencies:**
   ```bash
   yarn install

2. **Start development server:**
   ```bash
   yarn start
   
3. **Run test suites:**
   ```bash
   yarn test
